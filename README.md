# Description
It is a repo of my .files
## Usage:

- git clone https://github.com/kensi-rus/dotfiles ~/.aux 
- Then go to ~/.aux/scripts and exec the ./setup_symlinks.sh<br>
  It will replace your current zsh, tmux and nano configs with symlinks to an actual files in ~/.aux/config folder.

#### It has embedded functions conf2git and git2conf for further push and pull configs.
